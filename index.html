Find the odd int KATA from Codewars: Given an array of integers, find the one that appears an odd number of times. There will always be only one integer that appears an odd number of times. 


funktion findOdd(A){
  if (A.length == 1 || A.indexOf(A[0],1) === -1){
      return A[0];
  } else {
    A.splice(A.indexOf(A[0],1), 1);
    A.shift();
    return findOdd(A);
  }
}
